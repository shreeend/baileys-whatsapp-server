WhatsApp API Server

This server provides a REST API for WhatsApp messaging using the Baileys library.

API Endpoints

GET /health

Check if the server is running.

POST /send-message

Send a text message to a WhatsApp number.

POST /send-media

Send a media message (image, document, etc.) to a WhatsApp number.

POST /disconnect

Logout and disconnect a WhatsApp session.

GET /status/:deviceId

Check the connection status of a device.

Socket.IO Events

This server also supports real-time communication using Socket.IO for QR code scanning and session updates.
